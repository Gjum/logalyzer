Minecraft Log Analyzer
======================

BASH utility to analyze the online times of players by scanning all available `.log.gz` logfiles.

Installation
------------

`$ cd YOUR_SERVER_DIR && git clone https://github.com/Gjum/minecraft-log-analyzer.git`

Usage
-----

`$ ./scan-all`

--------------
